Crowd Funding
Seed Fund
Seed Fund is a fully decentralized application designed to allow users to create campaigns that can be funded by people worldwide using cryptocurrency. This approach maintains transparency and ensures that campaign creators have full control and access to their campaigns.

Krypt
Krypt is a fully decentralized crypto exchange where users can send and receive cryptocurrency from anyone, anywhere in the world. It ensures privacy, security, and transparency with a built-in ETH scan.
Legal
Vote Verse
Vote Verse is a decentralized application (DApp) that allows users to nominate themselves for campaigns and enables others to vote for them using their crypto wallets. It features real-time vote updates and high security through blockchain technology, making it more reliable than traditional voting methods.
Medical
De-Medical Records
De-Medical Records is a decentralized application (DApp) designed to allow hospitals to manage medical records and files in a decentralized manner, while also enabling users to keep their health records secure and private using blockchain technology.
